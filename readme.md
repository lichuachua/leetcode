# 数组：

数组类型的题目大致可以分为一下几种类型：

- 1.找规律、模拟类型
- 2.排序类型
- 3.二分法（折半查找）类型
- 4.双指针类型
- 5.滑动窗口类型

# 链表：

链表类型的题目大致可以分为一下几种类型：

- 1.按顺序遍历，双指针
- 2.链表反转（全部反转，区间反转，分组反转，找中点反转）
- 3.转为数组遍历（栈的思想），map存储
- 4.判断环，找倒数结点快慢指针
- 5.链表排序，主要是归并排序，双指针

# 栈

栈类型题目大致可以分为一下几种类型：

- 1.普通栈（配对类题目）
- 2.单调栈（找最大，下一个最大）
